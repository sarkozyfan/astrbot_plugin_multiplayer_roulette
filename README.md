# 恶魔轮盘赌

基于恶魔轮盘赌规则的 QQ 群游戏 AstrBot 插件。

## 功能

- 自定义人数：`/轮盘 开始 @玩家1 @玩家2`
- 弹药机制：实弹、空包弹、随机装填
- 回合机制：空包打自己继续回合，其余射击结算后切换回合
- 道具系统：放大镜、啤酒、手铐、香烟、锯子、逆转器、手机、过期药、注射器
- 道具上限：每名玩家最多持有 8 个道具
- 战绩系统：胜场、败场、开枪数、命中数、排行榜
- 配置面板：超时、血量范围、装弹范围、道具数量、AI 触发

## 命令

```text
/轮盘 开始 @玩家1 @玩家2
/轮盘 自己
/轮盘 对方 @玩家
/轮盘 道具
/轮盘 使用 放大镜
/轮盘 使用 手铐 @玩家
/轮盘 使用 注射器 @玩家
/轮盘 状态
/轮盘 战绩
/轮盘 排行榜
/轮盘 配置
/轮盘 结束
/轮盘 帮助
```

## 道具

- 放大镜：查看当前第一发是实弹还是空包弹
- 啤酒：退掉当前第一发，并公开弹种
- 手铐：指定一名存活玩家，跳过其下一次回合
- 香烟：回复 1 点 HP，不超过上限
- 锯子：下一发实弹伤害 +1，空包不会消耗效果
- 逆转器：把当前第一发实弹和空包弹互换，不显示逆转后的结果
- 手机：查看随机位置的一发弹药
- 过期药：50% 回复 2 点 HP，50% 损失 1 点 HP
- 注射器：抢夺一名存活玩家的随机一个道具，目标需要有道具

## 配置

以下项目可在 AstrBot 插件面板中调整：

- `roulette_timeout_seconds`：房间操作超时时间
- `roulette_min_hp` / `roulette_max_hp`：每局随机初始 HP 范围
- `roulette_min_shells` / `roulette_max_shells`：每轮装填弹药数范围
- `roulette_min_live_shells`：每轮装填保底实弹数
- `roulette_items_per_round`：每轮每名存活玩家获得的道具数
- `roulette_allow_ai_trigger`：是否允许 AI 触发
- `ai_trigger_delay`：AI 触发延迟
